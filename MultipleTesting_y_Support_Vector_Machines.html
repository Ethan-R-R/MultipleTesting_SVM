Ahora que se tienen las 10 variables con mayor diferencia de promedio se van a hacer mltiples pruebas de hiptesis para ver cules son las variables que tienen importancia estadstica.

Estas pruebas de hiptesis se harn con el estadstico T de student, con la hiptesis nula de que el promedio de cada variable entre ambas muestras es igual para las dos categoras.

Para esto se utilizar la funcin multipletests de statsmodels.stats.multitest. A cada gen se le hizo una prueba de hiptesis y todo se junt en un DataFrame.

Despus de eso, se ajustaron los errores para evitar falsos descubrimientos. Se compararn los resultados de utilizar tres metodologas diferentes, metodologa de Bonferroni, de Holm y de Benjamini Hochberg.

Ahora se har un ANOVA unidireccional. Esto prueba la hiptesis nula de que dos o ms grupos comparten la media. Primero se van a separar por clases los genes que quedan (de manera similar a como se hizo con las clases 2 y 4) despus se har el ANOVA utilizando la funcin f_oneway para cada gen. Finalmente, se har la correccin de pvalues con metodologa de Bonferroni, de Holm y de Benjamini Hochberg.

#Kernel lineal
%matplotlib inline
import matplotlib.pyplot as plt

from sklearn.metrics import classification_report, confusion_matrix, ConfusionMatrixDisplay
print("Kernel Lineal")
linealModel = SVC(kernel = 'linear').fit(X_train,Y_train)
yhat = linealModel.predict(X_test)
print(classification_report(Y_test,yhat))
ConfusionMatrixDisplay(confusion_matrix(Y_test,yhat)).plot()

print("Kernel Polinomial")
polyModel = SVC(kernel = 'poly',degree = 3).fit(X_train,Y_train)
yhat2 = polyModel.predict(X_test)
print(classification_report(Y_test,yhat2))
ConfusionMatrixDisplay(confusion_matrix(Y_test,yhat2)).plot()
plt.show()

print("Kernel Radial")
radialModel = SVC(kernel = 'rbf').fit(X_train,Y_train)
yhat3 = radialModel.predict(X_test)
print(classification_report(Y_test,yhat3))
ConfusionMatrixDisplay(confusion_matrix(Y_test,yhat3)).plot()
plt.show()
